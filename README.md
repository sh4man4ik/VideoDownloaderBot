# What is it?

This is a Telegram Bot that helps you download videos from any social network. All you have to do is send the bot a link
to the video and the bot will send you the video.

# Quick Start

The Telegram Bot can be accessed at https://t.me/video_downloa6er_bot

# Source Code

Don't forget to create a _.env_ file and put the bot token (BOT_TOKEN) in it.

Clone the repository:

```
git clone https://github.com/sh4man4ik/VideoDownloaderBot.git
```

Go to the directory:

```
cd VideoDownloaderBot
```

Download the dependencies:

```
npm install
```

Launch the bot:

```
node src/script.js
```

# Technology

The following technologies are used to develop the bot: _JavaScript_, _Node.js_.
